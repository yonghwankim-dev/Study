문제를 해결하였다. 문제의 원인은 어떤 한 페이지 그룹의 첫번째 숫자를 계산할때 마지막 숫자에서 한번에 보여줄 페이지의 수를 뺀것이 문제였다. 결국은 이전 페이지 그룹의 값을
이용하여 문제를 해결하였다. 페이지네이션은 익숙한 주제였지만 페이지 구현에 필요한 값들의 도출 과정이 헷갈리는 부분이 있어서 오늘 내일해서 글쓰기를 하고 기회가 된다면 스쿼드
세션에서 발표하기로 생각했다.
