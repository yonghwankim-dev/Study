## 무료 도메인 생성하기
freenom 플랫폼을 이용하여 무료 도메인을 생성해봅니다.
- link : https://www.freenom.com/en/index.html?lang=en

freenom 사이트에 접속한 다음에 무료 도메인 가입을 진행합니다.
- 현재 기술적 문제로 무료 도메인을 선택할 수 없습니다.
