H-Index : 과학자의 생산성과 영향력을 나타내는 지표.
어떤 과학자가 발표한 논문 n편 중, h회 이상 인용된 논문이 h편이상이고,
나머지 논문이 h번 이하 인용되었을때,
h의 최댓값

전체 논문의 인용횟수를 가진 배열에서 "전체 n편 논문중 h편의 논문이 h회 이상 인용되었다"
조건에 부합하는 h를 찾아야한다.
이때 과학자가 발표한 전체 논문수 n과 H-Index가 될 수 있는 h의 관계는 h<=n이다.

1. 인용횟수 배열 정렬
"~이상인 요소의 개수"를 구해야하기 때문에 오름차순 정렬
(Q. 왜 오름차순으로 정렬해야하는가?)

2. 인용횟수 배열 요소 값을 차례로 h로 지정하며 H-Index 조건 여부 파악
(해당 요소의 논문 인용횟수 >= h회 이상 인용된 논문 개수) 조건에 부합하면 해당 차례에서 'h회 이상
인용된 논문 개수'가 H-Index이다.
'h회 이상 인용된 논문 개수' : 오름차순 정렬했을 때, 해당 요소부터 마지막 요소까지의 요소 개수

입력값이 {4, 0, 6, 1, 5}일경우, 오름차순은 {0, 1, 4, 5, 6,}이 된다.
여기서 각 요소값보다 인용횟수가 크거나 같은 논문 편수를 h로 잡고, H-index 조건을 체크해보면 조건에 부합하는
값중 최대값인 H-index는 3인걸 알 수 있습니다. (*h는 citations[i]보다 작거나 같은 수)

해당논문 인용 횟수 = citations[i]
해당 논문보다 인용횟수가 크거나 같은 논문 편수 = h = n - i
조건 : h회이상 인용된 논문 편수가 h편 이상이다.


i=0, citations[0] = 0, h = 5, 5회이상 인용된 논문 편수가 5편이상이다. => false
i=1, citations[1] = 1, h = 4, 4회이상 인용된 논문 편수가 4편이상이다. => false
i=2, citations[2] = 4, h = 3, 3회이상 인용된 논문 편수가 3편이상이다. =? true, h-index = 3
i=3, citations[3] = 5, h = 2, 2회이상 인용된 논문 편수가 2편이상이다. => true
i=4, citations[4] = 6, h = 1, 1회이상 인용된 논문 편수가 1편이상이다. => true
